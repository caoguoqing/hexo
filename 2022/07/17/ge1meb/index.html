今天给大家带来一款免费的云服务器，配置是 4h32g，debian 系统。无需使用信用卡。

开始操作

第一步

先注册一个 github(如果有可跳过注册环节，注意：不要使用 QQ 邮箱注册的 github 账户)
https://github.com/MetaVPS/xssh
进入这个 github 仓库，点击 fork 按钮

进入”Create a new fork”后直接点击”Create”其他的都不用改

第二步

https://dashboard.ngrok.com/login
点开上面的链接，进入后直接使用 github 账号登陆，登陆完成后在侧边栏依次点击: Getting Started-Your Authtoken，然后复制左面的令牌，那一段长长的。然后回到自己 Fork 完的仓库，编辑”config.sh”，将刚刚复制的东西填到 NGROK_TOKEN=”你复制的”，仓库地址：https://github.com/你的用户名/xssh。

第三步

https://cloud.okteto.com/#/login
点开上面的网站，同样使用 github 的账号登录，登陆好后，点击”Launch Dev Environment”，再 From 选择 Git URL，之后在”Repository URL”填写上你的仓库链接，格式：https://github.com/你的名字/xssh，随后，在”Branch”填写”main”，填写完后点击 Launch 按钮，等一段时间，就会显示 Running。

第四部

然后我们回到第二步给的网站，在侧边栏依次点击 Cloud Edge-Endpoints，点击后右边后有
这个，然后点击它，在右边会弹出一个窗口，其中有个 URL，以我的这个为例，tcp://6.tcp.ngrok.io:17065，其中，6.tcp.ngrok.io 是这台服务器的 IP 地址，17065 是这台服务器的端口。

第五步

这样服务器就到手了，使用 ssh 连接工具连接这台服务器，在上步说了 IP 地址和端口是哪几部分，

用户名：root
密码：majalaya

之后就可以连接成功，这样就白嫖成功了。不要存放重要的资料
感谢 github、ngrok、okteto
